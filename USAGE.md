# 快速使用指南

## 🚀 立即开始（无需 API Key）

1. 用 **Chrome** 或 **Edge** 浏览器打开 `index.html`
2. 点击麦克风按钮，开始说话
3. 待办事项自动生成！

就是这么简单！无需任何配置，完全免费使用。

## 💬 语音输入示例

### 单个任务
```
说："明天要给客户发邮件"
→ 生成：给客户发邮件（截止：明天）
```

### 多个任务
```
说："今天要买菜，明天开会，后天交报告"
→ 生成 3 个待办事项，自动识别日期
```

### 带优先级
```
说："紧急！今天必须完成季度总结"
→ 生成：完成季度总结（优先级：高，截止：今天）
```

### 复杂任务
```
说："明天给张三打电话，还有周五之前完成项目报告"
→ 生成 2 个待办：
  1. 给张三打电话（明天）
  2. 完成项目报告（周五）
```

## 🎯 使用技巧

### 1. 智能编辑确认（新功能！）
语音识别完成后，会弹出确认框：
- 点击"确定"：直接处理识别结果
- 点击"取消"：可以手动编辑修正识别错误

**可在设置中关闭此功能，实现识别后直接处理**

### 2. 智能文本润色（新功能！）
自动优化口语化表达，让待办更简洁专业！

**润色示例**：
- 原文："明天要去开会"
  - 低强度：开会
  - 中强度：开会
  - 高强度：开会

- 原文："记得帮我把报告搞定"
  - 低强度：帮我把报告搞定
  - 中强度：把报告搞定
  - 高强度：完成报告

**润色强度设置**（在设置中调整）：
- **低**：仅移除时间词和优先级词，保持原样
- **中**：移除常见口语词，适度优化（默认，推荐）
- **高**：深度润色，口语转书面语

### 3. 时间识别
支持的时间表达：
- "今天"、"明天"、"后天"
- "周一"、"周二"...（自动识别本周或下周）
- "本周五"、"下周三"

### 4. 优先级识别
高优先级关键词：**紧急**、**必须**、**重要**、**马上**、**立刻**
低优先级关键词：**有空**、**可以**、**想要**、**打算**

### 5. 多任务分隔
用以下词语分隔多个任务：
- 逗号、顿号
- "还有"、"然后"、"接着"、"另外"

## ⚙️ 两种模式

### 简单模式（默认，免费）
✅ 完全免费，无需任何 API Key
✅ 使用规则提取待办事项
✅ 支持时间识别、优先级判断
✅ 适合日常使用

**特点**：
- 语音识别后直接处理文本
- 按逗号、"还有"等分隔符拆分任务
- 自动识别时间词（明天、周五等）
- 自动识别优先级词（紧急、重要等）

### AI 智能模式（可选，需 API Key）
✨ 更准确的任务提取
✨ 理解复杂的口语表达
✨ 更智能的优先级判断

**启用方法**：
1. 点击右上角"设置"
2. 选择"设置 API Key"
3. 输入 Claude API Key（从 https://console.anthropic.com 获取）

## 📝 管理待办

- **完成**：点击 checkbox，待办移到"已完成"区域
- **编辑**：点击"编辑"按钮修改内容
- **删除**：点击"删除"按钮
- **排序**：自动按优先级排序（高 → 中 → 低）

## 💾 数据存储

- 所有数据保存在浏览器本地（localStorage）
- 关闭浏览器后数据仍然保留
- 可以在设置中导出数据备份

## 🔧 设置功能

点击右上角"⚙️ 设置"可以：
1. **设置/修改/移除 API Key** - 切换简单模式和 AI 智能模式
2. **偏好设置** ⭐ 新功能
   - **编辑确认**：开启/关闭识别后的编辑功能
   - **润色强度**：低/中/高，控制文本优化程度
3. **清空所有待办** - 删除所有数据
4. **导出数据** - 备份为 JSON 格式

## ⚠️ 注意事项

1. **浏览器限制**：只支持 Chrome 和 Edge
2. **麦克风权限**：首次使用需要允许麦克风访问
3. **网络要求**：语音识别需要网络连接
4. **清晰发音**：在安静环境下，清晰地说话

## 🆚 模式对比

| 功能 | 简单模式 | AI 智能模式 |
|------|---------|------------|
| 费用 | 免费 | 需要 API Key |
| 任务提取 | 规则匹配 | AI 理解 |
| 准确度 | 良好 | 更高 |
| 复杂语句 | 一般 | 很好 |

## 💡 建议

- **日常使用**：简单模式完全够用
- **复杂需求**：如果经常使用复杂的口语表达，可以考虑 AI 模式
- **先试用**：建议先用简单模式，觉得不够用再开启 AI
