# 测试用例 - 验证文本处理准确度

## 🎯 测试目标

验证语音识别后的文本处理是否准确，包括：
1. 任务分割是否正确
2. 时间提取是否准确
3. 文本润色是否恰当（不改变原意）

---

## 📝 测试用例

### 用例 1：简单单任务
**输入**：
```
"明天要去开会"
```

**期望结果**：
- 任务数：1个
- 标题：
  - 低强度：去开会
  - 中强度：去开会
  - 高强度：开会
- 截止日期：明天
- 优先级：中

---

### 用例 2：多任务分割
**输入**：
```
"今天要买菜，明天开会，还有周五交报告"
```

**期望结果**：
- 任务数：3个
- 任务1：买菜（今天）
- 任务2：开会（明天）
- 任务3：交报告（周五）

---

### 用例 3：复杂多任务
**输入**：
```
"明天上午10点开会，下午要给客户打电话，还要完成项目文档"
```

**期望结果**：
- 任务数：3个
- 任务1：开会（明天）
- 任务2：给客户打电话（明天）
- 任务3：完成项目文档（明天）

---

### 用例 4：带优先级
**输入**：
```
"紧急！今天必须完成季度总结"
```

**期望结果**：
- 任务数：1个
- 标题：
  - 低强度：完成季度总结
  - 中强度：完成季度总结
  - 高强度：完成季度总结
- 截止日期：今天
- 优先级：高

---

### 用例 5：口语化表达（中强度）
**输入**：
```
"记得帮我把那个报告搞定"
```

**期望结果**：
- 低强度：帮我把那个报告搞定
- 中强度：把那个报告搞定
- 高强度：完成那个报告

---

### 用例 6：不应该错误分割
**输入**：
```
"买菜，做饭，洗碗"
```

**期望结果**：
- 任务数：1个（因为都是相关的连续动作）
- 标题：买菜，做饭，洗碗

**或者**：
- 任务数：3个（如果用户意图是分开的任务）

**实际处理**：
- 因为没有时间词，会被识别为一个任务的描述

---

### 用例 7：时间识别准确度
**输入**：
```
"本周五之前完成项目，下周一开始新任务"
```

**期望结果**：
- 任务数：2个
- 任务1：完成项目（本周五）
- 任务2：开始新任务（下周一）

---

### 用例 8：避免过度清理
**输入**：
```
"给张三打电话"
```

**期望结果**：
- 标题：
  - 低强度：张三打电话
  - 中强度：张三打电话
  - 高强度：向张三打电话

**错误示例**（应避免）：
- ❌ "张三打" （过度清理，丢失"电话"）
- ❌ "打电话" （丢失"张三"）

---

### 用例 9：保留关键信息
**输入**：
```
"跟小王讨论项目方案"
```

**期望结果**：
- 标题：
  - 低强度：跟小王讨论项目方案
  - 中强度：跟小王讨论项目方案
  - 高强度：和小王讨论项目方案

**不应该**：
- ❌ "讨论项目方案"（丢失"小王"）

---

### 用例 10：处理口头禅
**输入**：
```
"嗯，明天要去开会，还有那个报告要完成"
```

**期望结果**：
- 任务数：2个
- 任务1：去开会（明天）
- 任务2：报告要完成 / 完成报告

**清理效果**：
- ✅ 移除开头的"嗯"
- ✅ "那个"可以保留或移除

---

## 🔧 如何测试

### 1. 设置润色强度
```
设置 → 偏好设置 → 选择润色强度（低/中/高）
```

### 2. 测试流程
1. 点击录音按钮
2. 说出测试用例的内容
3. 点击停止
4. 查看生成的待办事项

### 3. 检查点
- [ ] 任务数量是否正确
- [ ] 任务标题是否准确（没有丢失关键信息）
- [ ] 时间提取是否正确
- [ ] 优先级判断是否合理
- [ ] 润色是否恰当（没有改变原意）

## ⚠️ 已知问题和注意事项

### 可能的问题
1. **过度分割**：
   - 问题："买菜，做饭" 被分成2个任务
   - 原因：逗号分隔符
   - 解决：用"和"连接 - "买菜和做饭"

2. **信息丢失**：
   - 问题："给张三打电话" 变成 "打电话"
   - 原因：过度润色
   - 解决：降低润色强度

3. **时间识别错误**：
   - 问题："下下周" 无法识别
   - 原因：未支持复杂时间表达
   - 解决：说"XX月XX日"

### 使用建议
1. **任务连接词**：
   - 使用 "还有"、"另外" 明确分隔任务
   - 避免单独使用逗号

2. **时间表达**：
   - 使用 "今天"、"明天"、"周X"
   - 避免 "一会儿"、"马上" 等模糊词

3. **润色强度选择**：
   - 个人日常：中强度
   - 工作正式：高强度
   - 保持原味：低强度
